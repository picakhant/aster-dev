# 🎬 Movie Database Web App (ရုပ်ရှင်ရှာဖွေရေး ပလက်ဖောင်း)

## 🚀 ပရောဂျက်အကြောင်း (Overview)
**Movie Database Web App** ဆိုတာက အသုံးပြုသူတွေအနေနဲ့ ခေတ်စားနေတဲ့ ရုပ်ရှင်တွေ၊ ကိုယ်နှစ်သက်တဲ့ ဇာတ်ကားတွေကို အလွယ်တကူ ရှာဖွေနိုင်ပြီး၊ နောက်ထွက်မယ့် ဇာတ်ကားသစ်တွေကိုပါ ခြေရာခံနိုင်မယ့် Full-stack Web Application တစ်ခု ဖြစ်ပါတယ်။ အသုံးပြုသူတွေအတွက် အကောင်းဆုံး အတွေ့အကြုံ (User Experience) ရစေဖို့ ခေတ်မီ နည်းပညာတွေကို အသုံးပြုပြီး ရေးသားထားပါတယ်။

## ✨ အဓိက ပါဝင်တဲ့ လုပ်ဆောင်ချက်များ (Key Features)
* **Dynamic Routing:** ဇာတ်ကားတစ်ကားချင်းစီရဲ့ အချက်အလက်တွေကို သီးသန့်ကြည့်နိုင်ဖို့ Next.js ကိုသုံးပြီး Dynamic Pages တွေ ဖန်တီးထားပါတယ်။
* **Real-time Data Fetching:** နောက်ဆုံးထွက် ရုပ်ရှင်ဒေတာတွေ၊ Rating တွေနဲ့ Trailer တွေကို အချိန်နဲ့တပြေးညီ ရယူနိုင်ဖို့ [TMDB API](https://www.themoviedb.org/) နဲ့ ချိတ်ဆက်ထားပါတယ်။
* **Secure Authentication:** အသုံးပြုသူတွေ အကောင့်ဖွင့်တာ၊ Login ဝင်တာတွေကို လုံခြုံစိတ်ချစွာ လုပ်ဆောင်နိုင်ဖို့ Appwrite ရဲ့ Auth စနစ်ကို အသုံးပြုထားပါတယ်။
* **Responsive UI:** ဖုန်း၊ တက်ဘလက်၊ ကွန်ပျူတာ စတဲ့ မည်သည့် Device မျိုးမှာမဆို ကြည့်ကောင်းနေစေဖို့ Tailwind CSS နဲ့ DaisyUI ကို သုံးပြီး ရေးဆွဲထားပါတယ်။

## 🛠️ အသုံးပြုထားသော နည်းပညာများ (Tech Stack)
ဒီပရောဂျက်မှာ Frontend (အမြင်ပိုင်း) နဲ့ Backend (နောက်ကွယ်က လုပ်ဆောင်ချက်ပိုင်း) ကို သေချာခွဲခြားပြီး (Decoupled Architecture) ရေးဆွဲထားပါတယ်။
* **Frontend:** React, Next.js 15, Tailwind CSS v4, DaisyUI
* **Backend:** Express, Node.js
* **BaaS:** Appwrite (Database, Storage, Auth)
* **External APIs:** TMDB API

## 🧠 အခက်အခဲများနှင့် ဖြေရှင်းချက်များ (Challenges & Solutions)
**အခက်အခဲ:** TMDB API ကနေ ပုံ (Image) အရေအတွက် အများကြီးကို လှမ်းဆွဲတဲ့အခါ Website ပွင့်လာတဲ့ အမြန်နှုန်း (Page Load Speed) ကျဆင်းသွားခြင်း။
**ဖြေရှင်းချက်:** Next.js ရဲ့ `<Image />` component ကို ပြောင်းလဲအသုံးပြုခဲ့ပြီး၊ ပုံတွေကို အလိုအလျောက် Optimize လုပ်ပေးတာနဲ့ (Lazy Loading) စနစ်တွေကြောင့် Website ရဲ့ Performance ကို သိသိသာသာ မြန်ဆန်သွားစေခဲ့ပါတယ်။
